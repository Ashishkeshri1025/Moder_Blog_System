Modern Blog System

A **Modern Blog System** is a responsive and interactive blogging platform built using **HTML, CSS, Bootstrap 5, JavaScript, and jQuery**.  
It allows users to explore blog posts by categories, read trending articles, write new posts, and perform basic authentication actions (simulated using localStorage).
